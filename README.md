# shop_proto

### 내용 및 기능

- 프론트 분야를 적용해보기 위한 프로젝트입니다.
- 상품 페이지의 일부 기능을 구현한 후, 이를 프론트로 
생성해보았습니다.
- thymeleaf를 적용하여, 서버 기능 중 일부를
프론트 부분까지 구현하는 것을 목표로 하였습니다.

### 사용 스킬 혹은 지식

- Java
- Spring Boot
- JPA
- MySQL
- Thymeleaf

### 진행기간

- 2022.02.25~ 2022.04.04

- 개발 전체을 맡았습니다.
